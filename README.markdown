BuildbotDocker
==============

An Easy CI environment for development.

![basic deployment][1]
![CI_with test deployment][2]

Requirement
-----------
* Docker
* Docker-compose
* ssh-keys(optional if you're using private repository)

Run
---
* $ docker-compose up -d
